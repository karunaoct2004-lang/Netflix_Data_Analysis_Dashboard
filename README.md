 Netflix Data Analysis Dashboard

The Netflix Data Analysis Dashboard is a data visualization and analytics project developed using Microsoft Power BI.  
The main purpose of this project is to analyze Netflix's content library and extract meaningful insights about movies, TV shows, release trends, countries, and content categories.

This interactive dashboard helps users understand how Netflix content is distributed globally and how the platform has evolved over time.

Project Objectives

The key objectives of this project are:

- Analyze the Netflix content dataset.
- Understand the distribution of Movies and TV Shows.
- Identify countries contributing the most Netflix content.
- Analyze content growth trends based on release years.
- Discover popular Netflix categories and genres.
- Create an interactive and user-friendly dashboard using Power BI.

Technologies Used

| Technology | Purpose |
| Power BI Desktop | Dashboard creation and visualization |
| Netflix Dataset | Data source for analysis |
| Data Cleaning | Preparing data for visualization |
| Data Visualization | Creating charts and insights |

Dataset Information

The dataset contains information about Netflix titles including:

- Show ID
- Type (Movie / TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Listed Categories
- Description

The dataset is used to analyze Netflix’s content availability and trends.

Dashboard Features

1. Netflix Content Overview

The dashboard provides an overall view of Netflix content:

- Total number of titles analyzed: **8.807K**
- Total Movies: **6.131K**
- Total TV Shows: **2.676K**

This gives a quick understanding of Netflix’s content volume.

2. Movies vs TV Shows Analysis

A comparison between Movies and TV Shows:

- Movies represent **69.62%** of Netflix content.
- TV Shows represent **30.38%** of Netflix content.

This visualization helps identify the dominant content type available on Netflix.

3. Netflix Content by Country

This analysis shows Netflix's global content distribution.

It helps answer:

- Which countries produce more Netflix content?
- How diverse is Netflix’s international library?
- Which regions contribute significantly to Netflix?

4. Content Release Trend Analysis

This section analyzes content growth based on release year.

Insights:

- Shows how Netflix content has increased over time.
- Helps identify periods of rapid content expansion.
- Displays release trends from older titles to recent content.

5. Top Netflix Categories

The dashboard analyzes the most common content categories.

Popular categories include:

- Dramas
- International Movies
- Documentaries
- Stand-Up Comedy
- Comedies
- Kids' TV
- Children & Family content

This helps understand audience preferences.
